## Server Components

Acho que a maioria de vocês acompanhou a atualização do Next JS nos útimos meses, chegando na versão 14
e trazendo muitas features e novidades no desenvolvimento web moderna. Uma das novas features são os Server Components do React.Js.

Acredito que para entender toda essa mudança que está acontecendo na web, como por exemplo os Server Components, precisamos mudar nosso modelo mental e
pensar de uma forma diferente ao buscar nossos dados.

Atualmente, no Next JS há três estratégias diferentes de renderização no servidor:
